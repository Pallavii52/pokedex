# Pokédex Application

A full-stack Pokédex application built using Spring Boot and React that allows users to search Pokémon and view detailed information using the PokeAPI.

## Features
- Search Pokémon by name
- View Pokémon details and stats
- RESTful API integration
- Caching for faster responses
- Responsive and interactive UI
- Error handling and validation

## Tech Stack
### Backend
- Java
- Spring Boot
- Spring Cache / Caffeine

### Frontend
- HTML
- JavaScript
- CSS

## API Used
- PokeAPI  
  https://pokeapi.co/
